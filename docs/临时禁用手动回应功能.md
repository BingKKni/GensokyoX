# 临时禁用手动回应功能

如果需要临时禁用我们新增的手动回应功能，可以按以下方式操作：

## 方法1：注释代码（推荐）

### send_group_msg.go
找到第35-59行，在所有代码前加 `//` 注释：

```go
// // 处理交互回应（按钮回调手动回应）
// if message.Params.InteractionID != nil && message.Params.InteractionID != "" {
//     // ... 所有手动回应代码都注释掉
// }
```

### send_private_msg.go  
找到第24-48行，同样注释掉所有手动回应代码。

## 方法2：修改判断条件

将判断条件改为永远不触发：

```go
// 临时禁用手动回应
if false && message.Params.InteractionID != nil && message.Params.InteractionID != "" {
    // ... 保持原有代码
}
```

## 重启程序

修改后需要重启程序才能生效。

## 恢复功能

需要恢复时，将注释去掉或将 `false` 条件删除即可。
